📖 相続税調査分析システム v10.0(職場ポリシー対応版) — 使い方ガイド

🔰 はじめに

本ツールは、相続税調査に必要な「取引履歴」「家族構成」「住所履歴」を統合分析し、資金移動や原資不明取引・名義預金の疑いを検出します。

本バージョンは職場ポリシー対応のため、以下の機能を意図的に無効化しています:

  • ❌ 外部ファイルの直接読み込み(CSV/TSV/Excel ファイル)
  • ❌ ブラウザ永続ストレージへの保存(自動保存なし)
  • ❌ クリップボードAPIによる自動コピー
  • ❌ ファイル自動保存機能
  • ❌ 外部通信API/外部CDN

すべての処理はブラウザ内メモリ上でのみ実行され、ページを閉じると入力データは完全に消去されます。

📥 データの入れ方

  • ① Excel からコピー → 貼り付け:左上セルをクリックして Ctrl+V で TSV を一括展開(自動コピーではなくユーザー操作によるOSクリップボード経由)
  • ② 直接入力:セルをクリックして手入力

📊 取引データ(13列)

銀行名 / 支店名 / 氏名 / 科目 / 口座番号 / 日付 / 時刻 / 出金 / 入金 / 取扱店 / 機番 / 摘要 / 残高

👨‍👩‍👧‍👦 家族データ(6列、必須)

氏名 / 続柄 / 生年月日 / 相続開始日 / 親の氏名 / 婚姻日

※ 続柄に「被相続人」「本人」「死亡者」「故人」のいずれかを含む行が必要です。その行の 相続開始日(D列)から相続開始日を抽出します。

🏠 住所データ(3列、任意)

氏名 / 住所 / 居住開始日

※ 住所データは v10.0 から任意 です。空欄でも取引・家族のみの簡易モードで分析可能です(住所移転関連の分析項目はスキップされます)。

📆 日付の入力形式

以下のすべてに対応します(自動正規化):

  • 西暦:2024-1-1 / 2024/01/01 / 2024.1.1
  • 和暦:R6.1.1 / 令和6.1.1 / H30.5.1 / 令和元.5.1
  • 取引データの省略年(1/15 等)は「取引期間」設定から推定

⌨️ 主なキーボードショートカット

  • Tab / Enter :次のセルへ
  • Shift+Tab :前のセルへ
  • Ctrl+C / Ctrl+V :コピー / 貼り付け(TSV 形式、OS標準クリップボード)
  • Delete :選択セルをクリア
  • 右クリック :コピー / 貼付 / 行挿入 などのメニュー

🔍 分析の流れ

  1. 取引・家族の2グリッド(最低)にデータを入れる(住所は任意)
  2. 分析設定(大額閾値、相続前後期間など)を調整
  3. 「📊 分析開始」をクリック
  4. 各タブで結果を確認。「⚠️ エラー一覧」タブで入力不備をチェック
  5. 必要に応じて「📋 画面内テキスト出力」または「🖨️ 印刷用表示」で結果を取得

📤 結果の取り出し方

  • 画面内テキスト出力:CSV風テキストが画面内モーダルに表示されます。Ctrl+A→Ctrl+C で手動コピーしてください(自動コピーは行いません)
  • 印刷用表示:ブラウザの印刷機能を呼び出します。「PDFとして保存」を選べばPDF化も可能(自動ダウンロードは行いません)

💾 データの保存について

v10.0 では自動保存を完全に停止しています。入力内容はブラウザ上のメモリ上にのみ保持され、ページを閉じる/再読み込みすると失われます。長時間の作業中はブラウザを閉じないよう注意してください。
